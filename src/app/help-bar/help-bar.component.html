BaoBab supports the reading of any content under the /var/www/html/api/content directory.
1.) To read a single article open the navigation tree and select the article in the tree.
2.) Alternatively select a group of articles in the tree, and select a single article from the section list
3.) Use the browser forward and back buttons to navigate between selected items.
 All BaoBab content is readable by all users, but only registered users
can modify site material.Before editing documents you must login to the site.
To edit a document:
1.) View a document by selecting it in the navigation tree or from the 'File/Read' menu
2.) Select 'File/Update' from the main menu and inline edit the document
3.) Press the 'Update' button to save edits or the 'Cancel' button to lose edits
 All BaoBab content is readable by all users, but only registered users can modify
site material. Before creating new documents you must login to the site.
To create a new document:
1.) Select 'File/Create' from the main menu.
2.) Enter the path of the document in the edit box
3.) Enter the content of the document in the text box.
4.) Press 'Create' to create the new document or 'Cancel' to lose edits
