BaoBab supports the reading of any content under a top level content directory.
1.) To read a single article open the navigation tree and select the article in the tree.
2.) Alternatively select a group of articles in the tree, and select a single article from the section list
3.) Use the browser forward and back buttons to navigate between selected items.
 All BaoBab content is readable by all users, but only registered users
can modify site material. Before editing articles you must login to the site.
To edit an article:
1.) View an article by selecting it in the navigation tree
2.) Edit the article
3.) Right click on the article in the tree view and select the menu item 'Save Article'
3.) Press 'Yes' in the confirmation dialog to save the article
 All BaoBab content is readable by all users, but only registered users can modify
site material. Before creating new articles you must login to the site.
To create a new article:
1.) Right click on a directory and select menu item 'New Article'
2.) In the 'Create' dialog enter a unique name of the new article and press 'Ok'
3.) The new article is created straightaway and highlighted in the tree.
4.) Start editing the new article and save details regularly (See help topic 'Editing Articles')
 All BaoBab content is readable by all users, but only registered users can modify
site material. Before deleting articles you must login to the site.
To delete an article:
1.) View an article by selecting it in the navigation tree
2.) Right click on the article in the tree view and select the menu item 'Delete Article'
3.) Press 'Yes' in the confirmation dialog to delete the article
 All BaoBab content is readable by all users, but only registered users can modify
site material. Before deleting directories you must login to the site.
To create a new directory:
1.) Right click on a directory and select menu item 'New Directory'
2.) In the 'Create' dialog enter a unique name of the new directory and press 'Ok'
3.) The new directory is created
 All BaoBab content is readable by all users, but only registered users can modify
site material. Before deleting directories you must login to the site.
To delete a directory:
1.) Select the directory in the navigation tree
2.) Right click on the directory in the tree view and select the menu item 'Delete Directory'
3.) Press 'Yes' in the confirmation dialog to delete the directory
 All BaoBab content is readable by all users, but only registered users can modify
site material. Before renaming articles you must login to the site.
To rename an article
1.) Select the article in the navigation tree
2.) Right click on the article in the tree view and select the menu item 'Rename Article'
3.) Rename the article and press the save button
 All BaoBab content is readable by all users, but only registered users can modify
site material. Before moving articles you must login to the site. Also note that by default
the ability to move articles is disabled. To allow articles to be moved see 'Configuring User Settings'
To move an article
1.) Select the article in the navigation tree
2.) Drag the article and drop it in a different directory

 Before configuring user settings you must login to the site.
To display and configure user settings, select 'Settings' from the 'Logon' menu.
Each user has the ability to configure the following settings:
'Allow articles to be moved' (No)- Switch to 'Yes' to allow articles to be moved via drag and drop.
'Auto save article' (No) - Switch to 'Yes' for current article to be automatically
saved without prompting. Set 'Save article every ? seconds' to set the
approximate frequency that articles will be automatically saved
'Auto save article before logout' (Yes)- Switch to 'Yes' to automatically save article
without prompting just before logout. Set 'Save ? seconds before logout' to set the
approxmimate time before logout that the article will be saved.
To save settings close the dialog by pressing the 'Save' button.
To view settings only and lose any changed settings, close the dialog by pressing the 'Cancel' button.
